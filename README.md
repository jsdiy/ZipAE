# ZIPアーカイバ（習作）
DotNetZip(Ionic.Zip)を利用したアーカイバ。習作です。
DotNetZipの更新は何年も止まっていましたが、2025年に.NET9対応で更新版が公開されました。
2013年に旧DotNetZipで作成した当アプリは、.NET Framework3.5とかC#7.0あたりの環境で作成されたものです。
それを、DotNetZipの更新を機に、
- .NET10
- Null許容型
- ラムダ式
- async-await

に慣れることを目的として、現在の開発環境でモダンなコードで作成し直しました。

# 特徴
## 二重起動時の処理
当アプリを起動した状態で、デスクトップに作ったショートカットへファイル/フォルダをD&Dした場合、
二重起動を検知し、先行起動しているインスタンスへ対象ファイル/フォルダのリストをプロセス間通信で送信し、
二重起動側は終了します。同時に、先行起動側ではWndProc()でリストを取得し、非同期で圧縮/展開処理を実行します。
この一連の流れを、なるべく厳密に処理するよう記述しています。

## 展開/圧縮の出力
展開/圧縮の出力先に同名フォルダ/同名ファイル(.zip)があった場合、
フォルダ名(2), ファイル名(2).zip のように「カッコ＋数字」で出力するようにしています。
「同名ファイルがあります。上書きしますか？」とダイアログを出してユーザーに一手間掛けさせるのが優れたUXとは思えません。
